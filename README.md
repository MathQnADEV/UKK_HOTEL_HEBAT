## Run Locally

Clone the project

```bash
  git clone https://github.com/MathQnADEV/UKK_HOTEL_HEBAT
```

Go to the project directory

```bash
  cd UKK_HOTEL_HEBAT
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
